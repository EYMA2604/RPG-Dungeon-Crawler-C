# Text-Based RPG Dungeon Crawler

A complex game engine written in C, focusing on manual memory management, generic data structures, and modular design.

## Key Features
* **Generic Data Structures:** Developed a custom **Binary Search Tree (BST)** using `void*` and function pointers to handle multiple entity types (Monsters/Items).
* **Manual Memory Management:** Implemented precise dynamic allocation and recursive cleanup to ensure **zero memory leaks**.
* **Spatial Map Engine:** Created a coordinate-based navigation system with automated map rendering in the console.
* **Game Mechanics:** Includes a turn-based combat system, inventory management, and path-finding logic.

## Tech Stack
* **Language:** C (C11)
* **Concepts:** Pointers, Recursion, Dynamic Memory (malloc/free), Modular Programming.
